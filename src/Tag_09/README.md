> Lernziele

> - Components

**Was passiert, wenn du auf einer normalen Website auf einen Link klickst?**
- es wird ein HTTP request erzeugt -> ein komplettes HTML Dokument wird geladen
- SPA: das klicken auf einen Link löst keinen HTTP Request aus
- SPA Seiten laden schneller und wirken wie eine Desptop APP (daher WEB APP)
