1. Sets (Einmalige Werte)

Ein Set speichert Werte so, dass jeder Wert nur ein einziges Mal vorkommen kann. Duplikate werden beim Hinzufügen einfach ignoriert.

Warum brauche ich das? Zum Beispiel für eine Liste von Tags unter einem Artikel oder um Unique-Besucher einer Webseite zu zählen, ohne dass doppelte Klicks die Statistik verfälschen.
