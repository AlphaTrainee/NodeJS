REST-API mit JSON-Support

Express.js REST-API mit JSON-Support und Datenbank

GET Requests

GET Alle Benutzer abrufen

Ruft alle Benutzer ab, optional mit Filtern.

GET Einzelnen Benutzer abrufen

Ruft einen spezifischen Benutzer anhand der ID ab.

POST Request

POST Neuen Benutzer erstellen

Erstellt einen neuen Benutzer mit JSON-Daten.

PUT Request

PUT Benutzer aktualisieren

Aktualisiert einen bestehenden Benutzer (partielle Updates möglich).

DELETE Request

DELETE Benutzer löschen

Löscht einen Benutzer anhand der ID.
